Connecting labels

Connecting labels: Unlabeled inputs

"59% of the 3.4 million form inputs identified were unlabled."

- The WebAIM Million

Connecting labels: Unlabeled inputs

<label>First Name</label>
<input type="text" />

Connecting labels: Explicit connection (visible)

<label for="firstName">First Name</label>
<input type="text" id="firstName" />

Connecting labels: Explicit connection (hidden)

<label for="search" className="visually-hidden">Search</label>
<input type="text" id="search" placeholder="Search" />

.visually-hidden {
  border: 0;
  clip: rect(0 0 0 0);
  height: 1px;
  margin: -1px;
  overflow: hidden;
  padding: 0;
  position: absolute;
  width: 1px;
}

Connecting labels: Implicit connection

<label>
  First Name
  <input type="text" />
</label>

Connecting labels: ARIA connection (visible)

First Name

<p id="firstName">First Name</P>
<input type="text" aria-labelledby="firstName" />

Connecting labels: ARIA connection (hidden)

<input type="text" aria-label="Search" placeholder="Search" />

Grouping inputs: When to group

Not all inputs need to be grouped. Group when you have:

  • Multiple choice questions
  • Multiple related fields

We will not spam you, or sell your email address.

  Email
